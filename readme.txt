Makes available shortcode for embed of single videos and grids from YouTube video playlists, which include schema.org markup as recommended by google.

== Description ==

Requires YouTube Data API key
shortcode examples
[yt_grid item_font_length=20 item_bg="#660000" item_spacing=10 yt_list_id="PL0FB9262CF878A34A"]
[yt_video_wrap width="50%" alignment="right" item_font_alignment=left]https://www.youtube.com/watch?v=qybUFnY7Y8w[/yt_video_wrap]
[yt_video video_id="qybUFnY7Y8w" width="100%" alignment="none"]
